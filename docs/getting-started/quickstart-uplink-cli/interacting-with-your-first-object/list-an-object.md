# List an Object

## List the object in our bucket

To view the cheesecake photo in our bucket, let's use the list command:

{% tabs %}
{% tab title="Windows" %}
```
./uplink.exe ls sj://cakes
```
{% endtab %}

{% tab title="Linux" %}
```
uplink ls sj://cakes
```
{% endtab %}

{% tab title="macOS" %}
```
uplink ls sj://cakes
```
{% endtab %}
{% endtabs %}
