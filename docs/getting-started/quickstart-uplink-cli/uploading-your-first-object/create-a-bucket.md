# Create a Bucket

Check [Prerequisites](../prerequisites.md).

## Create a bucket in our Project

Let's create a bucket to store photos of cake for our "food app" project:

{% tabs %}
{% tab title="Windows" %}
```
./uplink.exe mb sj://cakes
```
{% endtab %}

{% tab title="Linux" %}
```
uplink mb sj://cakes
```
{% endtab %}

{% tab title="macOS" %}
```
uplink mb sj://cakes
```
{% endtab %}
{% endtabs %}
